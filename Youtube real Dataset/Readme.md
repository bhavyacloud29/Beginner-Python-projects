# 📊 YouTube Global Data Analysis (Exploratory Data Analysis)

## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on a global YouTube dataset to uncover insights about popular YouTube channels across different countries.

The analysis explores trends in **subscribers, views, categories, and geographical distribution** of YouTube creators using Python data analysis libraries.

EDA helps in discovering patterns, detecting anomalies, and understanding relationships between variables before applying machine learning models.

---

# 📂 Dataset

The dataset contains information about global YouTube channels including:

- Channel name
- Country
- Category
- Subscriber count
- Total views
- Video count
- Channel rankings

File used in this project:

```
Global YouTube Statistics.csv
```

---

# 🎯 Objectives

The objectives of this project are:

- Analyze **global distribution of YouTube creators**
- Identify **top YouTubers worldwide**
- Study **subscriber growth trends**
- Compare **content categories**
- Understand **country-wise influence of YouTube creators**
- Visualize trends using charts and graphs

# 📈 Key Insights

Some important insights from the dataset:

- A small number of YouTube channels dominate subscriber counts.
- Entertainment and music categories have the largest audiences.
- Certain countries produce a higher number of popular YouTubers.
- Subscriber distribution is heavily skewed toward top creators.
