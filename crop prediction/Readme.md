# Crop Prediction using Machine Learning

This project predicts the most suitable crop based on soil nutrients and environmental conditions.

## Features
- Flask web interface
- Machine learning prediction
- Crop recommendation based on soil parameters

## Input Parameters
- Nitrogen
- Phosphorus
- Potassium
- Temperature
- Humidity
- pH
- Rainfall

## Output 
Predict the suitable crop to be planted

## Tech Stack
- Python
- Scikit-learn
- Flask
- HTML/CSS
